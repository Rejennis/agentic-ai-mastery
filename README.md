# Agentic AI Mastery

This repository contains the project structure for building agentic AI systems.

## Structure
- `agents/` — core agent roles (planner, executor, critic, router)
- `mcp/` — Model Context Protocol server/client components
- `memory/` — short-term and long-term memory modules
- `tools/` — reusable tools/functions used by agents
- `workflows/` — end-to-end workflows and orchestration
- `observability/` — logging, tracing, evaluation
- `deploy/` — deployment assets and configs
- `docs/` — design docs and notes

## Getting started
Add your implementations inside each folder. See `docs/design.md` for design notes.
